# templates

Os templates desta pasta definem a estrutura padrao usada no repositorio.

## Templates atuais

- [book-template](book-template): modelo enxuto para livros e referencias de base.
- [roadmap-template](roadmap-template): modelo completo para modulos de estudo com teoria, laboratorio e artigos.

## Regra de aplicacao

- Use `book-template` para fontes de referencia como livros.
- Use `roadmap-template` para trilhas de estudo e topicos praticos.

Manter esses modelos simples e consistentes evita proliferacao de estruturas diferentes no projeto.
