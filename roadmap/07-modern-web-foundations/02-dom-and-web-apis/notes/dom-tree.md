# Árvore de DOM

## O que é o DOM?

Document Object Model (DOM) é uma representação em árvore de um documento HTML ou XML. Cada elemento, atributo e texto no documento é representado como um nó na árvore. O DOM permite que os desenvolvedores acessem e manipulem a estrutura, estilo e conteúdo de um documento de forma programática usando linguagens de script como JavaScript.

o window.document é o ponto de entrada para acessar o DOM. Ele representa a janela do navegador e fornece métodos e propriedades para interagir com o conteúdo da página.

## Estrutura da Árvore de DOM

A árvore de DOM é composta por vários tipos de nós, incluindo:

- **Elementos**: Representam as tags HTML, como `<div>`, `<p>`, etc.
- **Atributos**: Representam os atributos das tags, como `class`, `id`, etc.
