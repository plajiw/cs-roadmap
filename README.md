Simulacao de um repositorio de estudos tecnico com foco em:

- navegacao humana simples
- uso facil por terminal
- contexto completo por tema

## Introducao curta

```c
#include <stdio.h>

int main(void)
{
        const char *descricao = "Repositorio de Estudos";
        printf("%s\n", descricao);
        return 0;
}
```

## Regras deste exemplo

- nomes de pastas em ingles
- anotacoes internas em portugues (PT-BR)
- organizacao principal em `roadmap/` por trilhas
- subtemas numerados dentro de cada trilha
- cursos, livros e praticas ficam dentro do proprio tema

## Estrutura principal

- `roadmap/`: trilhas e temas
- `resources/`: guias e templates de apoio transversal
- `scripts/`: automacao simples para criar e manter modulos
