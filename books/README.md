# books

Esta pasta concentra referencias-base de estudo que sustentam o roadmap principal. Ela nao replica a estrutura do `roadmap/`; funciona como biblioteca de apoio, com uma organizacao mais enxuta e focada em leitura, anotacao e consolidacao.

## Padrao de cada livro

Cada livro segue um template leve:

- `summary.md`: o que o livro cobre e por que ele importa.
- `links.md`: referencias do livro, edicao, autor ou materiais complementares.
- `notes/`: anotacoes, resumos de capitulos e insights praticos.

## Como usar

- Use `books/` como fonte de base teorica.
- Use `roadmap/` como trilha de aplicacao, pratica e consolidacao.
- Sempre que um livro apoiar diretamente uma trilha, registre essa conexao no `summary.md` do livro ou nas notas.
